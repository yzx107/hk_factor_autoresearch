# Gossip Intake Guide

## 这张表是干什么的

这张表是给同事记录“庄家/席位/风格 gossip”的。

目标不是证明谁真的在做，而是把当时听到的判断留痕，后面拿去看有没有交易 edge。

## 怎么填

- 一行填一条 gossip
- 先按当时判断填写，不要事后根据走势修改
- 不确定可以写模糊一点，但尽量别空太多

## 每列怎么理解

- `记录日期`
  - 你听到这条 gossip，或者决定开始跟踪这只票的日期

- `股票代码`
  - 五位代码

- `股票名称`
  - 方便人工阅读，可选但建议填

- `是否明确有人在做`
  - 建议填：`是 / 否 / 不确定`

- `你猜是谁在做`
  - 可以写人名、资金类型、圈内代号，或者模糊标签
  - 例：`operator_a`、`某家族办公室`、`小票资金群`

- `你猜常用什么席位`
  - 写你觉得更相关的席位、券商、通道、broker cluster

- `你猜是什么风格`
  - 建议直接用自然语言
  - 例：
    - `慢推`
    - `洗一下再拉`
    - `高开硬顶`
    - `尾盘托`
    - `冲高派发`

- `你的方向判断`
  - 建议填：`long / short / unclear`

- `你的把握`
  - 建议填：`高 / 中 / 低`

- `你觉得多久兑现`
  - 直接写数字加单位即可
  - 例：`3天`、`5天`、`2周`

- `是否需要盘面确认`
  - 建议填：`是 / 否`

- `你最主要的依据1/2/3`
  - 写最关键的几点，不用写很多
  - 例：
    - `盘口承接强`
    - `像老席位风格`
    - `前高附近不掉`
    - `拉升节奏像以前`

- `补充备注`
  - 写其他你觉得重要的信息

## 推荐填写原则

- 宁可写朴素一点，也不要为了标准化而失真
- 不知道就写“不确定”
- 尽量把“为什么这么觉得”写进依据或备注里
